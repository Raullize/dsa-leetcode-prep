# dsa-leetcode-prep

This repository contains my progress, code implementations, and notes for the Udemy course: [**JavaScript Data Structures & Algorithms + LeetCode Exercises**](https://www.udemy.com/course/data-structures-algorithms-javascript/).

## Course Overview

The course focuses on mastering Data Structures and Algorithms using JavaScript, with an emphasis on step-by-step animated walkthroughs and coding interview preparation.

## Repository Purpose

I am using this repository to document my learning process through:

* **Code Implementations:** Writing out data structures (such as Linked Lists, Trees, and Hash Tables) and algorithms from scratch in JavaScript.
* **LeetCode Solutions:** Solving practice problems to apply theoretical concepts to real-world interview scenarios.
* **Technical Notations:** Documenting the logic behind each solution, including Big O time and space complexity analysis.

## Organization

The repository is structured by topic to make it easy to reference specific concepts:

* **Data Structures:** Implementations of fundamental structures.
* **Algorithms:** Sorting, searching, and traversal methods.
* **LeetCode:** Solutions categorized by problem type and difficulty.
