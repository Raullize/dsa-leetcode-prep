# Classes & Pointers

This module covers the fundamental building blocks needed to implement custom data structures in JavaScript: **Classes** and **Pointers** (References).

## Overview

Understanding these concepts is crucial because almost all non-primitive data structures (like Linked Lists, Trees, and Graphs) are built using classes to define their structure and pointers to connect their elements.
